# 🏍️ Stealth Tech Portfolio & Interactive Winding Career Highway

Welcome to the repository for **Deepak Majhi's Software Engineering Portfolio**! 

Designed specifically for an SDE at Amazon, this is an ultra-premium, "stealth-tech" interactive landing page. It eschews generic grids and noisy neon glows in favor of a sleek, titanium-silver, carbon-black, and stealth-indigo dashboard that feels alive under the cursor.

Live Site: **[deepakmajhi22.github.io](https://deepakmajhi22.github.io/)**

## 🚀 Elite Features & Mechanics

### 1. 🏍️ The Winding Career Highway
* **Scroll-Linked Vector Physics**: A vertical winding highway track that spans a massive `2800px` depth (`3000px` on mobile/tablets).
* **Cafe Racer Animation**: Features an exquisitely detailed, top-view **Classic Cafe Racer** vector SVG complete with Boxer Twin engine cylinder fins, spoked wheels, clubman handlebars with circular bar-end mirrors, and dual megaphone exhausts.
* **Math-Coordinated Cornering**: Using dynamic scroll slope derivatives, the Cafe Racer leans up to **`8 degrees`** naturally into road curves, locking straight on upright stretches to ensure zero jitter.
* **Responsive Highway Timeline**: SDE Full-Time (10% depth), SDE Intern (40% depth), B.Tech ECE at NIT Jaipur (72% depth), separated by a long immersive empty winding stretch ("a bit road"). On mobile, it responsive-collapses into a linear vertical timeline.

### 2. 🌌 Interactive Stealth Cyber Grid Spotlight
* **Blueprint Spotlight**: A dynamic canvas mesh spotlight tracking the mouse cursor.
* **Interactive Blueprint Mesh**: Highly subtle horizontal and vertical blueprint gridlines (`48px` by `48px` cells) illuminate dynamically locally directly under the mouse cursor.

### 3. 💾 Fleeing Compiler Tokens (Magnetic Repulsion)
* **SDE Space Atmosphere**: Instead of typical space star dust, the atmosphere is populated with **`260` semi-transparent compiler tokens** (`0`, `1`, `{`, `}`, `[]`, `++`, `&&`) rendered in a clean `11px JetBrains Mono` font.
* **Cursor Repulsion Field**: Upgraded attraction physics to a high-velocity **magnetic repulsion field** (radius `180px`, force `8.0`). As your cursor sweeps across the screen, code elements scatter away dynamically in real time, leaving a wide, clean bubble tracking your cursor.

### 4. ✉️ Secure Contact Portal (FormSubmit.co AJAX Routing)
* **Real-World Integration**: Modern, secure out-of-the-box contact form using the asynchronous Fetch API to send encrypted payloads straight to `dmdeepakmajhi@gmail.com` via **FormSubmit.co**.
* **Zero Redirect Page Flow**: Employs custom visual "Transmitting..." console log traces, locks buttons during transmission, and displays an elegant glassmorphic success modal overlay upon completion.
* **Regional Comedy Easter Egg**: Uses legendary Tollywood comedian Brahmanandam's character **"Gajala (CEO of Sonic Solutions, Washington D.C.)"** and `gajala@sonicsolutions.com` as interactive placeholders!

## 🛠️ The Tech Stack

* **Structure**: Semantic HTML5 (with direct Font Awesome 6 icons integration, white/black Amazon Smile badges, and inline vector mathematics).
* **Styling**: Vanilla CSS3 (curated HSL dark/light modes, glassmorphism, responsive grid timeline queries, custom button animations).
* **Logic**: Vanilla ES6+ Javascript (IntersectionObserver hooks, dynamic canvas paint loops, real-time scroll lean physics, asynchronous fetch requests).

## 💻 Local Quickstart

1. **Clone the repository**:
   ```bash
   git clone https://github.com/deepakmajhi22/deepak-portfolio.git
   cd deepak-portfolio
   ```

2. **Run a local server**:
   You can spin up a quick Python local server to test features:
   ```bash
   python3 -m http.server 8000
   ```
   Open `http://localhost:8000` in your web browser.
